RDFa Test Suite

Overview

The RDFa Test Harness provides an interactive mechanism to check the conformance of an RDFa extractor and SPARQL query engine. This service is intended for anybody that wants to see how RDFa works as well as developers that want to do conformance testing on their own RDFa/SPARQL implementations.

There are several things that you can do on this web page:

  • To run a unit test, click on the [TEST] link.
  • To see the details of a unit test, click on the [show details] link.
  • You can execute every unit test by clicking on the [Run All Unit Tests] button.
  • You can modify the Test Suite, Approval Status, RDFa Extractor Service and SPARQL Engine URLs.

Status

  • The latest RDFa Implementation Report is available at the W3C site.
  • These tests are in development; until a test is approved by the RDF in XHTML Taskforce, it is subject to change without notice.
  • All bugs, suggestions and thoughts about The RDFa Test Harness (Crazy Ivan) can be directed towards Manu Sporny on the RDFa task force mailing list.
